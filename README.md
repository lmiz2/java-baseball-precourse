# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
 - 출력 기능
 - 입력 기능
 - 게임 라이프사이클 관리 기능
   - 프로그램 시작
   - 게임 시작
   - 게임 종료
     - 게임 종료 조건을 만족할때까지 반복
   - 프로그램 종료(terminate)
 - 정답 생성 기능
   - 임의의수 가져오기 기능
 - 입력결과 평가 기능
   - 스트라이크
   - n볼
   - Not thing
